### Introduction

**_SRI FTSensor SDK_** is a C++ wrappper of the FT Sensor made by Sunrise Instruments.

### Dependencies

* Boost.Asio

### Usage

```bash
$ git clone https://github.com/thinkexist1989/SRI-FTSensor-SDK.git
$ cd SRI-FTSensor-SDK/
$ mkdir build && cd build
$ cmake ..
$ make -j`nproc`
```
